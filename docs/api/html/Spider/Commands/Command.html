Command

class Command implements CommandInterface

Command: Contains the command script to be executed by the driver

For instance, a Command object may contain the OrientDB query "SELECT FROM users WHERE username = 'michael'"

You may create a Command explicitly or use the Command Builder and a driver specific Command Processor.
